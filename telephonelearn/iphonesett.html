Paramètres iphone

Nous allons voir dans l'ordre l'utilité de tous ces paramètres.

  1. Mode avion : Lorsqu'il est activé, enlève tous les réseaux du téléphone, y compris la Wi-Fi
  2. Wi-Fi : Paramètre dans lequel vous vous connectez à un réseau local.
  3. Bluetooth : Dans ce paramètre vous pouvez activer/désactiver le réseau bluetooth
  4. Données cellulaires : Les données cellulaires sont le reseau internet propre à votre abonnement (à activer seulement si votre abonnement le permet)
  5. Partage de connexion : Lié aux données cellulaires pour partager votre connexion avec des amis ou sur votre ordinateur

  6. Notifications : Dans ce menu vous pouvez gérer les notifications que vous recevez.
  7. Sons et vibrations : Pour régler le volume des notifications/appels ainsi que la puissance des vibrations.
  8. Concentration : Permet de rentrer en mode "Concentration" ainsi désactiver certaines notifications et prévenir vos correspondants que vous êtes occupés
  9. Temps d'écran : Vous renseigne sur le temps que vous avez passer à regarder votre écran mais aussi de mettre des limites de temps sur certaines applications.

  10. Général : Une vaste liste de paramètres sont proposés dans cet onglet, notamment la langue du téléphone, la date, l'heure..
  11. Centre de contrôle : Gère les applications en raccourcis lorsque vous glissez votre doigt en haut à droite de l'écran.
  12. Luminosité et affichage : Dans ce menu vous pouvez augmenter et baisser la luminosité de votre écran ainsi que le temps avant que votre écran ne se verouille.
  13. Écran d'accueil :Permet de gérer où sont affichées les nouvelles applications téléchargées.

  15. Fond d'écran À partir de ce menu vous pouvez modifier l'image en arrière plan de votre écran, soit une photo personnelle soit un fond d'écran prédéfini
